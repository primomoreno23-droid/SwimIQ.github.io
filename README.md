### REPO OVERVIEW

This is a repo dedicated to the Devlog of Team Grouse's SwimIQ Senior inquiry project

### HOW TO ADD A NEW POST

1. Duplicate file posts/week4-project-overview.html
2. Rename it (e.g. posts/week5-feature-name.html)
3. Edit the content, date, author, and week badge inside
4. Copy the <a class="post-card"> block above this comment
5. Update href, badges, title, and excerpt
6. Paste the new card ABOVE the previous ones (newest first)
